Geochemical Plots

Justin R. Jent

2017-11-12

The manager packages provides functions for geochemical plots.

# load example data
data("gw_data")

gw_data %>%
  piper_plot()

gw_data %>%
  schoeller_plot()

gw_data %>%
  filter(location_id %in% c("MW-1", "MW-2")) %>%
  stiff_plot(., total_dissolved_solids = "Total Dissolved Solids")
